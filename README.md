# techfest-futurex
TECHFEST FUTUREX

A futuristic, immersive, and responsive landing page inspired by the innovation-driven atmosphere of Techfest IIT Bombay. This project showcases a next-generation UI experience using cyberpunk aesthetics, interactive animations, holographic visuals, AI-inspired sections, and modern web technologies.

🚀 Features
Futuristic Techfest-inspired UI/UX
Responsive design for desktop and mobile
Animated particle background
Neon glowing cursor effects
Typing text animation
Holographic image effects
Scroll reveal animations
Interactive AI chatbot section
Countdown timer
Glassmorphism design elements
Smooth scrolling navigation
Dynamic progress bar
Modern cyberpunk-inspired interface

🛠️ Technologies Used
HTML5
CSS3
JavaScript
Particles.js
Google Fonts (Orbitron)
Interactive user experience
Animated transitions and hover effects
Techfest-inspired visual identity
